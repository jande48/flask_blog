to install user authentication with flask

from the command line, install bcrypt

pipenv install flask-bcrypt

import Bcrypt into your __init__.py file

from flask_bcrypt import Bcrypt

in your __init__.py file, define bcrypt

bcrypt = Bcrpyt(app)

in your routes.py file, import the variable bcrypt from the package

from flaskblog import app, db, bcrpyt

in your register route, create a hashed password 

hashed_pw = bcrypt.generate_password_hash(form.password.data).decode('utf-8')
user = User(username=form.username.data, email=form.email.data, password=hashed_pw)
db.session.add(user)
